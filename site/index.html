What is HTML?

Hypertext Markup Language: Text which contains links to other text.
  • HTML, provides the structure.
  • CSS, Provides the style.
  • Javascript, provides behavior

History of HTML

Before 1997

Not driven by any standards. Many browsers not compatible with the website.

Around 1997

The World Wide Web Consortium created <HTML & 4.0 /4.01>

Around 2000

W3C came up with XHTML 1.0.

2004

WHATAWG: HTML "Web Hypertext Application Technology Group."

2007

WHATWG & W3C start to work together

2011

Which created what we have now HTML5

W3C in charge of the standard HTML5 WHATWG in charge of HTML evolving

Anatomy of an HTML TAG

Important to understand syntax

<p> content </p>

hr and br tags do not have a closing tag e.g. <hr> <br>

Attribute is a name value pair. <p id="myid"></p>

enclosing value in quotes is not required in all circumstances, but best practice to always surround value of attribute in '' or " ".

E.g. <p onclick="alert('hi')"></p>

these are all block level elements
