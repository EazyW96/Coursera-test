What is HTML?

Hypertext Markup Language: Text which contains links to other text.
  • HTML, provides the structure.
  • CSS, Provides the style.
  • Javascript, provides behavior

History of HTML

Before 1997, Not driven by any standards. Many browsers not compatible with the website.
Around 1997 The World Wide Web Consortium created <HTML & 4.0 /4.01>
Around 2000 W3C came up with XHTML 1.0.
2004: WHATAWG: HTML "Web Hypertext Application Technology Group."
2007: WHATWG & W3C start to work together
2011: which created what we have now HTML5

W3C in charge of the standard HTML5 WHATWG in charge of HTML evolving
